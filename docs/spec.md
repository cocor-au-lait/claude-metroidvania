# メトロイドヴァニア プロトタイプ 仕様書

## 概要

2D横スクロールの探索型アクションゲーム（メトロイドヴァニア）のプロトタイプ。
プレイヤーはアビリティを獲得しながらマップを探索し、ボスを倒してクリアを目指す。

## 技術スタック

| 項目 | 内容 |
|------|------|
| フレームワーク | Phaser 3.x |
| 言語 | TypeScript |
| ビルドツール | Vite |
| 実行環境 | ブラウザ (Web) |
| バージョン管理 | GitHub |
| パッケージ管理 | npm |

## ゲーム仕様（プロトタイプスコープ：最小限）

### プレイヤー

| 項目 | 内容 |
|------|------|
| 移動 | 左右移動、ジャンプ |
| 攻撃 | 近接攻撃（剣による斬撃） |
| HP | 3（ハート3個） |
| アビリティ | 初期なし → 二段ジャンプを1つ獲得 |

**操作方法**

| キー | 動作 |
|------|------|
| ← / → または A / D | 左右移動 |
| Space または Z | ジャンプ（二段ジャンプ取得後は2回押せる） |
| X または J | 攻撃 |

### マップ構成

1エリアで構成された探索マップ。
各部屋はスクロールで繋がり、ドアや通路で行き来できる。

```
[スタート部屋]──[通路]──[アビリティ部屋]
     │                        │
  [通路]                   [通路]
     │                        │
[中間エリア]──[通路]──[ボス部屋]
```

| 部屋 | 説明 |
|------|------|
| スタート部屋 | 初期出現位置、セーブポイントあり |
| 通路・中間エリア | 敵が配置された探索エリア |
| アビリティ部屋 | 二段ジャンプを入手できる |
| ボス部屋 | ボスとの戦闘エリア |

### 敵

| 種類 | 動作 | HP | ダメージ |
|------|------|----|---------|
| スライム（雑魚） | 左右に巡回、壁で折り返す | 2 | 1 |
| ボス | 左右移動 + ジャンプ攻撃 | 10 | 2 |

### アビリティ

| アビリティ | 入手場所 | 効果 |
|-----------|---------|------|
| 二段ジャンプ | アビリティ部屋 | 空中でもう一度ジャンプできる |

### セーブ・チェックポイント

- スタート部屋にセーブポイントが1か所
- 触れると HP 全回復 + リスポーン地点を記録
- ゲームオーバー時はセーブポイントから再開

### ゲームクリア・ゲームオーバー

| 条件 | 結果 |
|------|------|
| ボスを倒す | クリア画面を表示 |
| HP が 0 になる | ゲームオーバー画面 → セーブポイントから再開 |

## 参考・制約

- プロトタイプのためグラフィックは仮素材（矩形や色付きスプライト）で可
- サウンドは省略可
- タイルマップはコードで生成する（Tiled エディタ不使用）
- モバイル対応は対象外
