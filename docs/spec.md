# メトロイドヴァニア プロトタイプ 仕様書

## 概要

2D横スクロールの探索型アクションゲーム（メトロイドヴァニア）のプロトタイプ。
プレイヤーはアビリティを獲得しながらマップを探索し、ボスを倒してクリアを目指す。

## 技術スタック

| 項目 | 内容 |
|------|------|
| フレームワーク | Phaser 3.x |
| 言語 | TypeScript |
| ビルドツール | Vite |
| 実行環境 | ブラウザ (Web) |
| バージョン管理 | GitHub |
| パッケージ管理 | npm |

## ゲーム仕様（プロトタイプスコープ：最小限）

### プレイヤー

| 項目 | 内容 |
|------|------|
| 移動 | 左右移動、ジャンプ |
| 攻撃 | 近接攻撃（剣による斬撃） |
| HP | 3（ハート3個） |
| アビリティ | 初期なし → 二段ジャンプを1つ獲得 |

**操作方法**

| キー | 動作 |
|------|------|
| ← / → または A / D | 左右移動 |
| Space または Z | ジャンプ（二段ジャンプ取得後は2回押せる） |
| X または J | 攻撃 |

### マップ構成

1エリアで構成された探索マップ。スタート部屋をハブとし、各方向に探索エリアが広がる。

```
          [アビリティ部屋]
                ↑↓ 自由に行き来可能
[中間エリア] ←→ [スタート（ハブ）] ←→ [ボス前室]
  （探索・            （初期地点）         （ここで壁 = ダブルジャンプ必須）
   セーブ）                                        ↓ ダブルジャンプで超える
                                              [ボス部屋]
```

#### 部屋の役割

| 部屋 | 方向 | 役割 |
|------|------|------|
| スタート部屋 | 中央（ハブ） | 初期出現位置。全エリアへの起点 |
| アビリティ部屋 | スタート上 | 二段ジャンプを入手。難所なしで到達可能 |
| 中間エリア | スタート左 | 敵が多い探索ゾーン。セーブポイントあり |
| ボス前室 | スタート右 | ボス部屋への通路。ダブルジャンプ必須の段差がある |
| ボス部屋 | ボス前室下 | ボスとの戦闘エリア（入室トリガーあり、詳細は下記） |

#### レベルデザイン方針

**アビリティ必須ゲート型**を採用する。これはメトロイドヴァニアの根幹である「行けない場所を発見する → 能力を獲得 → 戻って解放される」というループを実現するための設計である。

- プレイヤーはスタートからボス前室を自由に探索できる
- ボス前室でダブルジャンプ必須の高台（3タイル以上）を視認するが越えられない
- アビリティ部屋でダブルジャンプを取得し、高台を越えてボス部屋へ到達する
- 中間エリアは「ボス前の立ち回り練習 + セーブポイント」として機能する

**ゲートの視認性**：高台は画面内から明らかに「届かない高さ」として認識できるよう設計し、プレイヤーが詰まったと感じずに自然と探索を促す。

> **「スキルなしでも戦えるが有利」型との比較**
> アビリティが1つ・ボスが1体のプロトタイプ規模では、オプション有利型はスキルの存在感が薄れやすい。必須ゲート型のほうがアビリティ取得の達成感が明確に出るため、本プロトタイプでは必須ゲート型を選択する。

#### ボス部屋の入室トリガー仕様

プレイヤーがボス部屋の入口を越えた瞬間をトリガーとして、以下を同時に行う。

1. **封鎖バリアの出現** — 入口に静的タイルを生成し、退路を塞ぐ
2. **ボス行動の開始** — トリガー前はボスを静止させ、トリガー後にパトロール＆ジャンプ攻撃を開始する

> **実装メモ（将来対応）**
> - トリガー検出: `physics.add.overlap` でセンサー矩形を使うか、毎フレームのプレイヤー座標監視
> - バリア: `platforms` グループに動的タイルを追加（`group.create(x, y, "tile-boss")`）して封鎖
> - ボス制御: `Boss` に `activate()` メソッドを用意し、待機中は `setVelocityX(0)` + パトロール・ジャンプを停止する

### 敵

| 種類 | 動作 | HP | ダメージ |
|------|------|----|---------|
| スライム（雑魚） | 左右に巡回、壁で折り返す | 2 | 1 |
| ボス | **入室トリガー後に**左右移動 + ジャンプ攻撃を開始（待機中は静止） | 10 | 2 |

### アビリティ

| アビリティ | 入手場所 | 効果 |
|-----------|---------|------|
| 二段ジャンプ | アビリティ部屋 | 空中でもう一度ジャンプできる |

### セーブ・チェックポイント

- **中間エリア**にセーブポイントが1か所（ボス挑戦前の最終セーブ地点として機能）
- 触れると HP 全回復 + リスポーン地点を記録
- ゲームオーバー時はセーブポイントから再開

### ゲームクリア・ゲームオーバー

| 条件 | 結果 |
|------|------|
| ボスを倒す | クリア画面を表示 |
| HP が 0 になる | ゲームオーバー画面 → セーブポイントから再開 |
